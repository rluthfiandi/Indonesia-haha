Perencanaan:
Penjelasan : Indonesia Events adalah website yang berisi tentang Event-Event di seluruh Indonesia yang mempunyai fitur sort by date atau by locations. 
Fungsi: Dapat memberikan informasi kepada semua orang mengenai event event yang ada di Indonesia
Tujuan: Untuk membantu mempromosikan Event agar bisa dilihat semua orang agar dapat terpublikasi dengan luas
Target pengguna: Semua orang
Kelebihan: Mencangkup semua Event di Indonesia dari Sabang sampai Marauke yang ingin mempromosikan atau mencari event di indonesia
Kekurangan : Untuk membeli ticket pengguna akan diarahkan ke website event itu sendiri
